# 네이밍 컨벤션

## 파일 및 디렉토리 네이밍

| 종류 | 규칙 | 예시 |
|------|------|------|
| 페이지 디렉토리 | kebab-case | `src/app/master/`, `src/app/complementary/` |
| 페이지 파일 | 고정명 `page.tsx` | `src/app/master/page.tsx` |
| 컴포넌트 파일 | PascalCase + `.tsx` | `OpacityGauge.tsx`, `InkDonutChart.tsx` |
| 라이브러리 파일 | camelCase + `.ts` | `api.ts`, `color.ts` |
| 레이아웃 파일 | 고정명 `layout.tsx` | `src/app/layout.tsx` |

## 컴포넌트 네이밍

```tsx
// ✅ PascalCase 컴포넌트명
export default function OpacityGauge({ value }: Props) { ... }
export default function InkDonutChart({ data }: Props) { ... }

// ❌ camelCase 컴포넌트명 금지
export default function opacityGauge() { ... }
```

## 인터페이스 및 타입

```tsx
// ✅ 인터페이스: PascalCase, 파일 상단에 정의
interface DashboardStats {
  total_inks: number
  avg_opacity: number
}

interface Props {
  value: number
  label: string
}

// ❌ type alias로 Props 정의하지 않음 (프로젝트 관례상 interface 사용)
type Props = { value: number }
```

## 변수 및 함수 네이밍

| 종류 | 규칙 | 예시 |
|------|------|------|
| 변수 | camelCase | `dashboardStats`, `activeTab` |
| 불리언 변수 | `is/has/show` 접두사 | `isLoading`, `hasError`, `showModal` |
| 이벤트 핸들러 | `handle` 접두사 | `handleSubmit`, `handleTabChange` |
| 커스텀 훅 | `use` 접두사 | `useInkData`, `useColorPreview` |
| 상수 | UPPER_SNAKE_CASE | `CATEGORY_LABELS`, `DELTA_E_THRESHOLDS` |

## import 경로

- 프로젝트 내부 모듈은 항상 `@/` 경로 별칭을 사용합니다.
- 상대 경로(`../`)는 사용하지 않습니다.

```tsx
// ✅ 경로 별칭 사용
import api from '@/lib/api'
import { labToRgb } from '@/lib/color'
import OpacityGauge from '@/components/OpacityGauge'

// ❌ 상대 경로 금지
import api from '../../lib/api'
import { labToRgb } from '../lib/color'
```

## export 패턴

```tsx
// ✅ 페이지 및 컴포넌트: default export
export default function MasterPage() { ... }
export default function OpacityGauge({ value }: Props) { ... }

// ✅ 유틸리티 함수: named export
export function labToRgb(L: number, a: number, b: number): string { ... }

// ❌ 페이지에 named export 사용 금지
export function MasterPage() { ... }
```

## 한국어 UI 텍스트

- UI에 표시되는 모든 텍스트는 한국어로 작성합니다.
- 변수명·함수명·타입명은 영어(카멜케이스·파스칼케이스)를 사용합니다.

```tsx
// ✅ UI 텍스트는 한국어
<button>← 메인</button>
<h1>잉크 마스터 관리</h1>

// ✅ 코드 식별자는 영어
const [activeTab, setActiveTab] = useState<string>('inks')
```

## 핵심 규칙 요약

| 규칙 | 내용 |
|------|------|
| 컴포넌트 파일 | PascalCase.tsx |
| 라이브러리 파일 | camelCase.ts |
| 페이지 경로 | kebab-case 디렉토리 + `page.tsx` |
| import 경로 | `@/` 별칭 필수 |
| export | 페이지·컴포넌트는 default, 유틸은 named |
| Props 타입 | `interface Props { }` |
| 상수 | UPPER_SNAKE_CASE |
