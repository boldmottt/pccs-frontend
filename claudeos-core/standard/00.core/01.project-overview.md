# 프로젝트 개요

## 프로젝트 정보

| 항목 | 내용 |
|------|------|
| 프로젝트명 | PCCS (Paint Color Control System) |
| 프레임워크 | Next.js App Router |
| 언어 | TypeScript 5 |
| 패키지 매니저 | npm |
| 개발 포트 | 8080 |
| UI 언어 | 한국어 |

## 기술 스택

| 범주 | 기술 |
|------|------|
| 프론트엔드 | Next.js App Router, TypeScript 5 |
| HTTP 클라이언트 | axios (싱글톤: `src/lib/api.ts`) |
| 스타일링 | 인라인 style 객체 (도메인 페이지), Tailwind CSS (홈 페이지) |
| 백엔드 | 외부 FastAPI (이 저장소에 포함되지 않음) |
| 테스트 | 없음 (미도입) |
| 상태 관리 | useState (로컬), 전역 상태 라이브러리 없음 |

## 라우팅 구조

App Router 방식을 사용합니다. 모든 도메인 페이지는 `'use client'` 지시어를 선언하며,
서버 컴포넌트(RSC)는 현재 사용되지 않습니다.

| 경로 | 파일 | 기능 |
|------|------|------|
| `/` | `src/app/page.tsx` | 홈(메인) — 도메인 탐색 |
| `/master` | `src/app/master/page.tsx` | 잉크 마스터 관리 (탭 CRUD) |
| `/dashboard` | `src/app/dashboard/page.tsx` | 통계 대시보드 |
| `/complementary` | `src/app/complementary/page.tsx` | 보색 추출 |

## 외부 백엔드 연동

모든 API 호출은 Next.js Route Handler를 우회하여 외부 FastAPI 백엔드로 직접 전송됩니다.

- **연결 지점**: `src/lib/api.ts` 의 axios 싱글톤
- **환경 변수**: `NEXT_PUBLIC_API_URL` (기본값: `http://localhost:8001`)
- **Next.js API Route**: 없음 — route.ts 파일이 존재하지 않습니다

## 배포 환경

- 개발: `npm run dev` (포트 8080)
- 프로덕션: `npm run build && npm run start`
- 빌드 결과물: `.next/`

## 현재 알려진 기술 부채

| 우선순위 | 항목 |
|----------|------|
| CRITICAL | `master/page.tsx` 에 `labToRgb` 함수 중복 정의 (`@/lib/color` 에서 import 해야 함) |
| CRITICAL | `master/page.tsx` 파일 크기 800줄 초과 |
| HIGH | form 상태 `useState<any>({})` — 타입 안전성 없음 |
| HIGH | `alert()` 로 오류 표시 — 인라인 에러 상태로 교체 필요 |
| HIGH | `error.tsx` / 에러 바운더리 없음 |
