# 아키텍처

## 전체 구조

```
[브라우저]
    │
    ▼
[Next.js App Router — src/app/]
    │  모든 페이지: 'use client'
    │  서버 컴포넌트(RSC) 없음
    │
    ▼
[공유 컴포넌트 — src/components/]
    OpacityGauge, InkDonutChart, TransparencySimulator
    │
    ▼
[유틸리티 레이어 — src/lib/]
    api.ts (axios 싱글톤)  ←→  [외부 FastAPI 백엔드]
    color.ts (labToRgb)
```

## 레이어 설명

### 페이지 레이어 (`src/app/<domain>/page.tsx`)

- 모든 도메인 페이지는 `'use client'` 지시어를 파일 맨 위에 선언합니다.
- `useEffect` + `useState` 로 데이터를 패칭하고 로딩 상태를 관리합니다.
- 도메인 내 비즈니스 로직(탭 상태, 폼 상태)을 직접 포함합니다.

### 컴포넌트 레이어 (`src/components/`)

- 여러 페이지에서 재사용되는 시각화 컴포넌트를 배치합니다.
- 파일 하나당 컴포넌트 하나 (배럴 index.tsx 없음).
- 공유 SVG 차트 컴포넌트: `OpacityGauge`, `InkDonutChart`.

### 유틸리티 레이어 (`src/lib/`)

- `api.ts`: `NEXT_PUBLIC_API_URL` 을 baseURL로 하는 axios 인스턴스. 모든 HTTP 호출의 진입점.
- `color.ts`: `labToRgb` 등 색상 변환 순수 함수.

## 응답 흐름

```
페이지 컴포넌트
  └─ useEffect
       └─ api.get('/endpoint')      ← src/lib/api.ts (axios 싱글톤)
            └─ 외부 FastAPI 백엔드
  └─ setState(data)
  └─ JSX 렌더링
```

**중요**: Next.js Route Handler(`route.ts`)나 Server Action은 존재하지 않습니다.
모든 API 호출은 클라이언트 → 외부 FastAPI 직접 통신입니다.

## 컴포넌트 계층 구조

```
src/app/layout.tsx              (루트 레이아웃, 정적 metadata)
├── src/app/page.tsx            (홈, Tailwind)
├── src/app/master/page.tsx     (잉크 마스터 — 탭 CRUD, SVG 시각화)
│   ├── <OpacityGauge />        (src/components/OpacityGauge.tsx)
│   ├── <InkDonutChart />       (src/components/InkDonutChart.tsx)
│   └── <TransparencySimulator />(src/components/TransparencySimulator.tsx)
├── src/app/dashboard/page.tsx  (대시보드 — CSS 바 차트)
└── src/app/complementary/page.tsx (보색 추출 — Promise.all 3-method)
```

## 주요 패턴

| 패턴 | 구현 |
|------|------|
| 데이터 패칭 | `useEffect` + `api.get()` (클라이언트 사이드) |
| 로딩 상태 | `useState<boolean>` (true/false 토글) |
| 에러 처리 | `try/catch` + `console.error` (UI 피드백 없음 — 개선 필요) |
| 네비게이션 | `router.push('/')` 로 메인 복귀 |
| 색상 미리보기 | `labToRgb` (from `@/lib/color`) 로 실시간 RGB 변환 |

## ✅ 올바른 예시

```tsx
'use client'
import { useEffect, useState } from 'react'
import api from '@/lib/api'
import { labToRgb } from '@/lib/color'

export default function DashboardPage() {
  const [data, setData] = useState<DashboardStats | null>(null)
  const [loading, setLoading] = useState(true)

  useEffect(() => {
    api.get('/dashboard/stats')
      .then(res => setData(res.data))
      .catch(err => console.error(err))
      .finally(() => setLoading(false))
  }, [])

  if (loading) return <div>로딩 중...</div>
  return <div>{/* 렌더링 */}</div>
}
```
