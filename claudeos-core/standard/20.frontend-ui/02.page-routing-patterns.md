# 페이지 및 라우팅 패턴

## App Router 구조

이 프로젝트는 Next.js App Router를 사용합니다.
도메인별로 `src/app/<domain>/page.tsx` 파일을 생성합니다.

```
src/app/
├── layout.tsx              # 루트 레이아웃 (공통 HTML, metadata)
├── page.tsx                # 홈 페이지 (/)
├── master/
│   └── page.tsx            # /master
├── dashboard/
│   └── page.tsx            # /dashboard
└── complementary/
    └── page.tsx            # /complementary
```

## 'use client' 지시어

모든 도메인 페이지는 파일 첫 줄에 `'use client'` 를 선언합니다.
서버 컴포넌트(RSC)와 Server Action은 현재 사용하지 않습니다.

```tsx
// ✅ 표준 페이지 파일 구조
'use client'

import { useEffect, useState } from 'react'
import { useRouter } from 'next/navigation'
import api from '@/lib/api'

interface DashboardStats {
  total_samples: number
  avg_delta_e: number
}

export default function DashboardPage() {
  const router = useRouter()
  const [stats, setStats] = useState<DashboardStats | null>(null)
  const [loading, setLoading] = useState(true)

  useEffect(() => {
    api.get('/dashboard/stats')
      .then(res => setStats(res.data))
      .catch(err => console.error(err))
      .finally(() => setLoading(false))
  }, [])

  return (
    <div>
      <button onClick={() => router.push('/')}>← 메인</button>
      {loading ? <p>로딩 중...</p> : <div>{/* 콘텐츠 */}</div>}
    </div>
  )
}
```

```tsx
// ❌ 금지 패턴
// Server Component에서 API 호출 — 이 프로젝트는 모두 'use client'
async function DashboardPage() {
  const data = await fetch('/api/stats')
  return <div>...</div>
}

// 'use client' 지시어 누락
import api from '@/lib/api'
export default function DashboardPage() { ... }
```

## 레이아웃

현재 도메인 수준의 `layout.tsx` 는 없습니다. 공통 레이아웃은 `src/app/layout.tsx` 만 존재합니다.

```tsx
// ✅ 루트 레이아웃 (src/app/layout.tsx)
export const metadata = {
  title: 'PCCS',
  description: 'Paint Color Control System',
}

export default function RootLayout({ children }: { children: React.ReactNode }) {
  return (
    <html lang="ko">
      <body>{children}</body>
    </html>
  )
}
```

## 네비게이션

페이지 내 이동은 `useRouter` 훅의 `router.push()` 를 사용합니다.
메인 페이지로의 복귀 버튼(`← 메인`)은 모든 도메인 페이지에 포함됩니다.

```tsx
// ✅ useRouter로 네비게이션
'use client'
import { useRouter } from 'next/navigation'

export default function MasterPage() {
  const router = useRouter()
  return (
    <button onClick={() => router.push('/')}>← 메인</button>
  )
}
```

```tsx
// ❌ next/router (Pages Router) 사용 금지 — App Router는 next/navigation 사용
import { useRouter } from 'next/router'
```

## 동적 라우트

현재 프로젝트에는 동적 라우트(`[id]`)가 없습니다.
신규 추가 시 `src/app/<domain>/[id]/page.tsx` 구조를 따릅니다.

## 미들웨어

현재 `middleware.ts` 가 없습니다. 인증 보호나 리디렉션이 필요하면
`src/middleware.ts` 에 추가합니다.

## 핵심 규칙 요약

| 규칙 | 내용 |
|------|------|
| 라우터 방식 | App Router (`src/app/<domain>/page.tsx`) |
| 지시어 | 모든 도메인 페이지 첫 줄에 `'use client'` |
| 네비게이션 훅 | `useRouter` from `next/navigation` |
| 레이아웃 | 루트 `layout.tsx` 하나만 존재 |
| 동적 라우트 | `[id]` 폴더 구조 (현재 미사용) |
| RSC/Server Action | 현재 미사용 |
